## 为什么选择MDX?

MDX 是 Markdown 的一种特殊变体，它支持嵌入 JavaScript 及其 JSX 语法。这使得用户能够将 JavaScript 与 UI组件结合到 Markdown 内容中，从而实现诸如交互式图表或警报等功能。

如果你已有使用 MDX 编写的现有内容，此集成有望使迁移到Astro变得轻而易举。
